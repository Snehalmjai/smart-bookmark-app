# Smart Bookmark App

A simple bookmark manager built with **Next.js (App Router)**, **Supabase (Auth, Database, Realtime)**, and **Tailwind CSS**.

## Features
- Google OAuth login (no email/password)
- Add bookmarks (URL + title)
- Private bookmarks per user (User A cannot see User Bâ€™s)
- Real-time updates across tabs (Supabase Realtime)
- Delete bookmarks
- Deployed on Vercel

## Tech Stack
- Next.js 16 (App Router)
- Supabase (Auth, Database, Realtime)
- Tailwind CSS v4
- Vercel (deployment)
